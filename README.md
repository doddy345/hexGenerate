Example Usage:
```
node ./dist/generateDirtSheet.js \
--base inputs/base.png \
--side inputs/side.png \
--sw3dSideBase inputs/sw3dside_brown.png \
--n3dSideBase inputs/n3dside_brown.png \
--nw3dSideBase inputs/nw3dside_brown.png \
--sw3dSideEdge inputs/sw3dside_green.png \
--n3dSideEdge inputs/n3dside_green.png \
--nw3dSideEdge inputs/nw3dside_green.png 
```
